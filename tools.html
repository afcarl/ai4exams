  • EQA Universal Evaluator

    A standardized evaluation script aids in establishing comparability between system results. We release the EQA Universal Evaluator as a method for computing standard evaluation measures on files in the EQA Universal Data Format. It also produces aggregate scores over multiple data sets (if provided). We include such meta-scores to help measure the general ability of each system, apart from any data-specific assumptions or biases.
